# Employee Portal

A simple React application where users log in, fill out an employee information form, and view a summary of the submitted details.

## Tech Stack

- React 18
- Vite
- React Router v6
- CSS Modules
- Vitest + Testing Library

## Prerequisites

Node.js 18 or higher

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

App runs at `http://localhost:5173`

## Tests

```bash
npm test
```

## Production Build

```bash
npm run build
```

## User Flow

1. **Login** — enter the credentials above
2. **Employee Form** — fill in employee details (name, department, role, etc.)
3. **Details View** — review the submitted information
4. **Logout** — return to the login screen
